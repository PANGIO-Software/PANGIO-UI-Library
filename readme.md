## Introduction
The **PANGIO UI Library** is a free library
for creating websites. It includes
basic CSS classes, as well as predefined classes for, e.g.,
buttons and forms. In addition, the library offers 
various JavaScript functions.

## Cascading Style Sheet
### Buttons
The following classes are applicable to the HTML elements 
`<button>` and `a`. The main class for **all** buttons is
`btn`.

- `btn-primary`
- `btn-success`
- `btn-warn`
- `btn-danger`

### Paddings
The predefined classes for padding range from `2px`
up to `16px`. There are five different naming patterns.
An example for padding across the entire element is 
`pangio-padding-10px`. For left-side padding, there 
is, for example, `pangio-padding-left-10px`. 
The remaining naming patterns follow the examples.
